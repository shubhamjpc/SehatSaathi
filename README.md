# SehatSaathi
SehatSaathi is a simple, low-bandwidth health assistant for rural India. Users can enter symptoms via text or voice to receive easy-to-understand guidance, safe home remedies, and alerts on when to seek medical care. Built for local languages, low literacy, and limited internet access.
